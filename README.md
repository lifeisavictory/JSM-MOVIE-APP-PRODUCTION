## 📌 O projektu

</br>

Movie aplikace postavená na **React + Vite** s integrací **TMDB API** a **Appwrite** databází. Projekt byl vytvořen jako součást kurzu **JavaScript Mastery**. Celkový Tailwind a kostra projektu poskytnuta na počátku kurzu.

> 🙏 Poděkování [JavaScript Mastery](https://www.jsmastery.pro) za tento kurz.

## 🚀 Funkce

</br>

| Funkce | Popis |
|---|---|
| 🔍 **Vyhledávání filmů** | Integrace s TMDB API pro vyhledávání napříč databází filmů |
| 📊 **Trendující filmy** | Sledování aktuálně populárních filmů |
| 💾 **Appwrite databáze** | Ukládání vyhledávacích metrik do cloudové databáze |
| 🎨 **Moderní UI** | Responzivní design postavený na Tailwind CSS |
| 🐳 **Docker podpora** | Snadné nasazení pomocí kontejnerizace |

## 🧩 Klíčové koncepty

</br>

| Koncept | Popis |
|---|---|
| 🪝 **React Hooks** | Práce s `useState` a `useEffect` |
| ⏱️ **useDebounce** | Vlastní hook zabraňující nadměrným voláním API |
| 🌐 **Async operace** | Správa více zdrojů dat a zpracování asynchronních operací |
| 🔗 **TMDB API** | Integrace s externím REST API pro data o filmech |
